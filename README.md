# David Paci

I build secure AI integrations and EU-hosted custom software for small and
mid-sized companies, including regulated industries.

Most of my work sits in the same place: a company already runs a stack that
works, and wants AI inside it rather than instead of it. That means retrieval
over messy internal knowledge, MCP servers that expose existing tools to
language models, and automation that leaves the surrounding systems alone.

Based in Villach, Austria. Everything I ship runs on EU infrastructure.

[Website](https://webse.at) · [LinkedIn](https://linkedin.com/in/DEIN-HANDLE) · [jukeep](https://jukeep.com) · [neurit.ai](https://neurit.ai)

## What I work on

- **Retrieval and knowledge systems.** Vector search, hybrid retrieval, and
  citation-backed answers over internal documents. The hard part is rarely the
  embedding model, it is what counts as a source and how you prove an answer.
- **MCP servers.** Exposing company data and internal tools to LLM clients in a
  way that is scoped, auditable, and boring to operate.
- **Custom software.** Next.js, Postgres with pgvector, Docker, self-hosted on
  Hetzner and netcup. No US-only dependencies in the critical path.
- **Process automation.** Multi-provider model routing, cost per operation as a
  hard constraint, and local models wherever a frontier model is overkill.
